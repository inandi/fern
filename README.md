# Project Mango

Turn your codebase into a clean, interactive map. **Scan once, navigate often.**

A VS Code extension that indexes PHP, JS, HTML, and CSS files and shows a tree of folders, files, classes, and methods. Click to jump to the definition; hover for docblocks.

## Run the extension

1. `npm install` then `npm run compile`
2. Press **F5** in VS Code to launch the Extension Development Host
3. Open a folder (e.g. a PHP or JS project), then click **Mango: Generate Map** in the status bar or run it from the Command Palette
4. The map opens in a new tab; click any symbol to open the file and go to the line
